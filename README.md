# KelTech Solutions Website

Official marketing website for **KelTech Solutions** — a technological digital solutions firm based in Mehsana/Visnagar.

**Tagline:** _Your Digital Assistant on one tap._

---

## What this website includes

- Home, About, Services, Team, Blog, and Contact sections
- WhatsApp floating chat button
- Google Maps office location embed
- Testimonials and FAQ sections
- Responsive design (mobile + desktop)
- Live 3D background animation (Three.js) + hover tilt interactions
- Contact form configured for **Netlify Forms**
- Thank-you page after successful contact form submission

---

## Project files

- `index.html` → Main website page
- `styles.css` → Styling, theme, animations, responsive layout
- `script.js` → Mobile menu + small interactive behaviors
- `thanks.html` → Redirect page shown after contact form submit
- `DEPLOYMENT_GUIDE.md` → Step-by-step launch guide (non-technical)

---

## How to run locally (quick preview)

### Option 1: Open directly
Double-click `index.html` in your computer.

### Option 2: Use local server (recommended)
If Python is installed:

```bash
python -m http.server 8080
```

Then open: `http://localhost:8080`
